# SKILL — page-change
**Layer 3.** Load only when a request is **PAGE-001** (add a self-contained block to an existing page).

## Inputs
- The content for the block (client-supplied or from an approved file).
- The target page + roughly where the block goes.

## Procedure
1. Confirm the source content exists and is approved (e.g. approved-testimonials).
2. Draft the block **in the client's voice**; keep it self-contained.
3. **Never restructure** surrounding content or navigation.
4. Specify placement precisely; save deliverable; write the note.

## Output contract
Additive only · approved content only · in voice · no structural/nav change · reversible.

## Stop conditions
- Requires new design or "make it look better" → 🔴 ESCALATE.
- Content not approved / not supplied → 🟡 HOLD.
- Needs nav or layout changes → 🔴 ESCALATE.
