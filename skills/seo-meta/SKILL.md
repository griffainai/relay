# SKILL — seo-meta
**Layer 3.** Load only when a request is **SEO-001** (write/improve a page's meta title, description, or alt text).

## Inputs
- The page and what it's about.
- The client's `CONTEXT.md` voice.

## Procedure
1. Write **one** option per element, in voice.
2. Respect limits: **title ≤ 60 chars**, **description ≤ 155 chars**, alt text concise + descriptive.
3. No keyword stuffing; read like a human.
4. Note the exact char count; save deliverable; write the note.

## Output contract
One option per element · within limits · in voice · char count shown · reversible.

## Stop conditions
- It's part of a "fix our SEO" project, not a specific page → 🔴 ESCALATE (scope).
- Page purpose unclear → 🟡 HOLD.
