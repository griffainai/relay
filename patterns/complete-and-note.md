# pattern — complete-and-note

**The move:** Every completed action produces a deliverable **and** a three-line note in the same breath. Work without a note didn't happen.

**Why:** The note is what lets a human trust autonomy. It turns "the AI did something" into "I can see exactly what changed, where, in ten seconds."

**Looks like:**
> deliverable → `## Completion note` (what changed · where · anything to glance at) → update STATE.

**Failure it prevents:** silent changes the team has to reverse-engineer — the fastest way to lose trust in an autonomous agent.
